# Agent Scoping Accuracy

An agent given a task can answer "which outcome does this serve?" and "what constraints apply?" without the user explaining it.

## Measurement
- Does the agent scope its work correctly without re-prompting?
- Compound effect visible after 5-10 sessions of accumulated metis
